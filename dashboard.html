Interior+: Dados, algoritmos e caminhos para a revitalização do interior de Portugal

Dada a existência de imensa informação desconectada ou perdida sobre o interior de Portugal, este projeto visou colmatar essa lacuna, agrupando e analisando dados das regiões interiores de Portugal, através de ciência de dados e análise textual. Este trabalho focou-se em três temáticas: o abandono rural, o desenvolvimento do interior e os programas de apoio existentes, utilizando os dados do Arquivo.pt para demonstrar padrões, soluções e eventuais caminhos para o melhoramento do interior através de uma abordagem baseada em automação e Ciência de Dados.
